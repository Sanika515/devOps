# 🧪 **Branch Creation, Merging, and Conflict Resolution in GitHub**

---

# 🎯 Aim

To create multiple branches, make changes, merge them, and resolve conflicts using Git and GitHub.

---

# 📘 Theory

### 🔹 Branch

A separate line of development used to work independently.

### 🔹 Merge

Combining changes from one branch into another.

### 🔹 Merge Conflict

Occurs when the same part of a file is modified in different branches.

---

### 🔹 Types of Merge

* **Fast-forward** → simple update
* **Three-way merge** → creates merge commit
* **Conflict merge** → requires manual resolution
