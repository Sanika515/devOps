🔷 Practical 20
Demonstrate container lifecycle management (start, stop, restart, remove) using Docker
🔹 Aim

To perform container lifecycle operations such as start, stop, restart, and remove using Docker.

🔹 Theory
🔸 Docker Container
A container is a running instance of a Docker image.
It allows applications to run in an isolated environment.
🔸 Container Lifecycle

A container goes through different states:

Created → container initialized
Running → actively executing
Stopped → paused/not running
Removed → deleted permanently
🔸 Lifecycle Commands
docker run → create & start container
docker stop → stop container
docker start → start stopped container
docker restart → restart container
docker rm → remove container
🔹 Requirements
Docker installed
Internet connection
